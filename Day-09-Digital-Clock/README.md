# Day 9: Neon Digital Clock & Stopwatch

## Features
- **Live Clock:** Updates every second.
- **Dual Format:** Toggle between 12h and 24h modes.
- **Stopwatch:** Start, Stop, and Reset functionality.
- **Neon Themes:** customizable colors (Green, Pink, Blue).

## Tech Stack
- HTML5
- CSS3 (Neon Glow Effects)
- JavaScript (Date Object & setInterval)
